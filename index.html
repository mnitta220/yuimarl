Yuimarl

「Yuimarl」（ゆいまーる）は、企業向けの情報システムです。以下の３つの目的を持っています。

  • 企業の業務を総合的に支援するシステムとなること。
    （現在は、人、組織、法人、物品の情報を管理する機能しかありません。）
  • 様々なシステムの“ひな型”となること。
    （Yuimarl をベースとすることで、迅速かつ低コストでの開発が可能となります。）
  • 情報システムの開発者に、Java EEの開発技術を普及すること。

Yuimarl の特長として、以下のような点を挙げることができます。

  • ソースコードが公開されており、誰でも無料で使用でき、変更も自由に行うことができます。
  • エンタープライズJavaの世界標準である、Java Platform, Enterprise Edition (Java EE)の最新バージョンが使用されています。
  • プログラム内部が、しっかりとした概念モデリングに基づく構造をしているため、再利用が容易です。
  • 実装済みの様々な機能部品を利用することができます。
  • スマートフォン、タブレットでアクセスすると、その画面に合わせたレイアウトになります。（レスポンシブデザイン）

名前の由来

「ゆいまーる」は、沖縄の言葉で、「相互扶助」（そうごふじょ）、「助け合い」を意味します。「ゆい」は「結ぶ」、「まーる」は「交流の場」です。このシステムが、多くの企業の助けになればよいと願っています。また、多くの方からのご意見、ご協力をいただき、このシステムが成長していけばよいと思っています。そうした願いと期待を込めて、この名前を付けました。

機能

現在は、人、組織、法人、物品の情報を管理する機能のみです。
今後は、スケジュール管理などの機能を追加していく予定です。（この予定は変更される可能性があります。）

実際の画面

次のアドレスにアクセスすると、Yuimarl を実際に使用することができます。（スマートフォン、タブレットでアクセスすると、その画面に合わせたレイアウトになります。）
https://www.bitfarm.co.jp/yuimarl-web/

以下のいずれかのユーザーID、パスワードを入力してログインしてください。

ユーザーID パスワード 説明
U00001 test すべての権限を持つシステム管理者。ユーザーを登録する権限を持つ。
U00002 test 人/組織/物、イベントの参照権限を持つ一般ユーザー。
U00003 test 何も権限を持たないユーザー。

データを自由に登録していただいて構いません。（登録されたデータは、定期的に元に戻されます。）

動作環境

Yuimarl の開発言語にはJava言語が使用されています。Javaを実行できる環境であれば、様々なOSで動作させることができます。データベースも様々なリレーショナルデータベース製品に対応させることが可能です。

アーキテクチャー

Yuimarl のシステムを構成するソフトウェア階層と、そこで使用されているプロダクトは、次のようになっています。

階層 プロダクト 説明
プレゼンテーション層 各種Webブラウザ Firefox、Google Chrome、Safari、Internet Explorer で動作確認済みです。
jQuery JavaScriptライブラリ。
jQuery UI jQueryを使用したユーザーインターフェースライブラリ。
CKEditor HTML形式で文章を入力するJavaScriptライブラリ。メモ入力画面で使用。
アプリケーション層 Java Runtime Environment Java実行環境。
Java EEアプリケーションサーバー Java Platform, Enterprise Edition（Java EE）の動作をサポートする実行環境。GlassFish、Payaraなど。
データアクセス層 各種データベース製品 現在は、 Java DB およびMySQLを使用。

カスタマイズ

Yuimarl のソースコードは、自由に改変することができます。Yuimarl をベースとして、別のシステムを開発することもできます。
カスタマイズを有限会社ビットファームに依頼される場合は、ご連絡ください。Yuimarl に取り込むかを検討させていただきます。内容によっては、有償での対応とさせていただくこともあります。

企業様として参加していただければ、それを、貴社としての強みにすることができます。生産性が向上し、競争力が高まります。
