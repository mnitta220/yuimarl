開発環境構築

前提条件

Yuimarl の開発を行うためには、以下のソフトウェアが必要です。

  • Git
  • Rust
  • gcloud CLI
  • VSCode
  • pug-cli

Google Cloud プロジェクトの作成

次のコマンドで、Yuimarl 用の Google Cloud プロジェクトを作成してください。（<project-id>の箇所は、作成するプロジェクトのID）

gcloud projects create <project-id>

プロジェクトを作成したら、次のコマンドを実行してください。

gcloud config set project <project-id>

gcloud auth application-default login

運用環境の構築

開発を行う前に、運用環境構築を実行して、開発用の Firebase と Cloud Run の環境を構築してください。

リポジトリのクローン

次のコマンドで、Yuimarl の Github リポジトリをクローンしてください。

git clone https://github.com/mnitta220/yuimarl.git

VSCode の起動

次のコマンドで、Yuimarl の VSCode のワークスペースを起動してください。

code workspace.code-workspace

.env の作成

プロジェクトのルートフォルダに、「.env.template」というファイルがあるので、それをコピーして、「.env」というファイルを作成してください。

「.env」に、運用環境構築の「Firebase SDK の追加」で表示された「firebaseConfig」の内容を設定してください。

Yuimarl の起動

次のコマンドで、Yuimarl を起動してください。

cargo run



この画像のように表示されたら、Web ブラウザで、http://localhost:8080にアクセスすると、Yuimarl のログイン画面が表示されます。

停止するには、Ctrl + C キーを押します。
