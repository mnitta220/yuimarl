データベース

コレクション

user

ユーザー情報を保持する。
項目名 型 説明
uid string
name string
email string
photo_url string
status number 0:未承認, 1:承認, 2:禁止
created_at timestamp
last_login timestamp

session

セッション情報を保持する。
項目名 型 説明
id string セッションID (uuid)
uid string ユーザーID
name string ユーザー名
email string
photo_url string
created_at timestamp

project

プロジェクト情報を保持する。
項目名 型 説明
id string プロジェクトID (uuid)
project_name string プロジェクト名
owner string プロジェクトオーナー（プロジェクトの作成者）
prefix string チケット接頭辞
language string 言語（"ja"固定）
ticket_limit string 最大チケット番号（1000固定）
ticket_number string チケット番号
created_at timestamp

project_member

プロジェクトのメンバー情報を保持する。
項目名 型 説明
id string (uuid)
project_id string プロジェクトID
member string メンバーのユーザーID
role number 1:オーナー, 2:管理者, 3:メンバー, 4:閲覧者
last_used timestamp

ticket

チケット情報を保持する。
項目名 型 説明
id string (uuid)
id_disp string 表示用チケットID（接頭辞＋連番）
project_id string プロジェクトID
name string チケット名
description string 内容
start_date string 開始日
end_date string 終了日
progress number 進捗率
priority string 優先度
parent string 親チケット
deliverables string 成果物(JSON)
owner string 登録ユーザー
history string 更新履歴(JSON)
created_at timestamp

ticket_note

チケットのノートの情報を保持する。
項目名 型 説明
id string チケットID
note string ノート（マークダウン）
history string 更新履歴(JSON)

ticket_member

チケットの担当者の情報を保持する。
項目名 型 説明
id string チケットID
uid string ユーザーID
seq number 表示順

news

ユーザーへのお知らせの情報を保持する。
項目名 型 説明
id string (uuid)
uid string ユーザーID
timestamp timestamp 日時
contents string お知らせ内容
