システムの仕様

ユーザー認証

以下のいずれかのメールアドレスまたはアカウントでログインする。

  • メールアドレス
  • Google
  • Facebook
  • X
  • Github
  • Apple
  • Yahoo!

利用規約に合意したら、システムのユーザーとして登録される。

プロジェクト

  • プロジェクトを作成したユーザーがオーナーになる。
  • 一人のユーザーが登録できるのは、最大で10個のプロジェクト。（他のユーザーが作成したプロジェクトには、何個でも参加できる）
  • 登録済みのユーザーを検索して、プロジェクトのメンバーとして登録することができる。メンバーには、オーナー/管理者/メンバー/閲覧者のいずれかのロールを付与する。
  • ロールが付与されていないユーザーは、閲覧も更新もできない。
  • １つのプロジェクトには、（オーナーを含めて）最大で20人のメンバーを追加することができる。
  • プロジェクトに21人以上のユーザーを追加したい場合は、有料での対応となる。

チケット

  • チケットには、接頭辞＋連番のIDが付与される。
  • １つのプロジェクトには、最大で1000個のチケットを作成することができる。（削除したチケットも1個とカウントされる）
  • 1001個以上のチケットを作成したい場合は、有料での対応となる。
